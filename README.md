# Base Pac-Man: Professional Farcaster Mini-App 🔵🕹️

A high-performance HTML5 Canvas implementation of Pac-Man, optimized for the **Farcaster v2 SDK** and **Base Ecosystem**.

## 🚀 Key Features
- **Farcaster Integration:** Seamless "Launch Frame" support within Warpcast.
- **On-chain Readiness:** Designed for Talent Protocol project verification.
- **Optimized Performance:** Lightweight vanilla JS engine for mobile experience.

## 🛠️ Technical Specifications
- **Framework:** Vanilla JavaScript / HTML5 Canvas
- **Style:** Tailwind CSS 3.0
- **SDK:** Farcaster Mini-App SDK v2
